老师是wbw老师，讲得很好。

平时分，看考勤、课堂展示和大作业。考勤，中间查了两三次；课堂展示，找靠谱队友，提前两周找老师确定选题和方向，打分权完全在老师，最高直接给到满分；大作业，老师的话说，比较看重你是否读出了书中“讲故事的方式”，本人理解，就是讲清楚本书作者的史观，描述这段历史的观点、立场。

期末考试，感觉这门课比较靠背，大部分答案都是死的，只要背了就能拿分。

选择题：考前突击过了一遍网上找到的一个[考研资料](http://szkb.wsyu.edu.cn/wcm.files/upload/CMSszkb/202304/202304131050045.pdf)，没想到最后选择还押中了不少题。

大题：基本都来自课本上的课后题，老师最后一节课有划重点，只要把标准答案背过就没问题。

个人建议至少留出2天复习时间吧。当时考完淑芬开始准备，基本也就2天，一天看选择知识点，一天背答题，就差不多。
