## 🌊 从“三文鱼卷”开始的旅途

“三文鱼卷”是由10系学长创立的微信公众号，曾经是覆盖航类、计类、医工班等学院的学习资源分享平台，帮助过无数伙伴（包括笔者），但是在2024年受不可抗力停止更新。（[最后一篇公众号](https://mp.weixin.qq.com/s/eyuAGtSxbU5LPHF02o3x_Q)）我们希望能够把学习与分享的火种传递下去，于是便有了这个仓库——“四文鱼卷”。

它保持三文鱼卷的初心不变，但是转移了阵地。为什么不继续使用微信公众号？因为卑鄙的微信约束，一个身份证只能建立一个个人公众号，而我已经建过了。更重要的是，GitHub 提供了**开放协作、去中心化、开源共享**的天然土壤。这意味着，即使我们这群老登毕业之后，任何小登都可以继续为这个仓库贡献资料，传递下去。

GitHub的维护更新并不复杂，可以理解为就是一个简单的免费云盘，在当前界面就可以像在云盘中一样下载、浏览仓库里的内容。如果为仓库贡献力量，只需要学习简单的 `Pull Request` 操作就可以。网络上有大量教程，在此不做赘述。学会使用GitHub进行版本管理、协作、查找开源项目，也是每一位10系学子必要掌握的技能。不如，就从这里开始试试吧！

## 📂 内容概览

目前本知识花园正在逐步建设中，已收录的学科资料如下：
- **[分子生物学](./分子生物学/)**：包含期末真题回忆等资料。
- **[生物信息学](./生物信息学/)**：包含历次实验课作业（R语言）代码及其相关分析数据。

- **[遗传学](./遗传学/)**：包含名词解释笔记、词汇自检清单及期末考试回忆等。

## 📜 仓库原则与收录指南

为了维护这片知识花园的纯净与长远发展，我们设立了以下收录原则：

### ✅ 我们欢迎并收集：
- 📝 **考试回忆**：历年真题回忆与期末重点梳理
- 💡 **经验分享**：学习方法、备考经验与心得体会
- 📚 **学习笔记**：优质的个人课堂笔记、总结与思维导图

### 🚫 我们不接受：
- ⚠️ 涉及版权保护的官方教材、原版专著及电子书等
- ⚠️ 未经授课教师允许外传的课件、PPT 、课程资料等
